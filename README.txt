To set up you need PHP7.
phpunit is in composer which requires a composer to be installed and and update.


 To run from command line type:
 php src/index.php example-input.txt 27/08/19 16:00 NW43QB 20


 where 27/08/19 is the date of arrival of the goods
 16:00 is time time of arrival

 your postcode is (no spaces) NW34BQ

 and the number of hungry people is 20

 this should return somehting like:
 "Premium meat selection;;"
 "Breakfast;gluten,eggs;"
